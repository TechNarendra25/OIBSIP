## 📌 Project Overview

This project was completed as part of the **Oasis Infobyte Data Science Internship (OIBSIP)**.

The objective is to develop a Machine Learning classification model that predicts the species of an Iris flower using its physical measurements.

The dataset is available directly in the **Scikit-learn** library.

---

# 🎯 Objective

Build and compare multiple Machine Learning classification algorithms to accurately classify Iris flowers into:

- 🌸 Iris Setosa
- 🌼 Iris Versicolor
- 🌺 Iris Virginica

---

# 📊 Dataset Information

- Total Samples: **150**
- Features: **4**
- Classes: **3**
- Dataset Source: `sklearn.datasets.load_iris()`

### Features

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)
